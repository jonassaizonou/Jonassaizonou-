# Portfolio Professionnel – Christian G. J. Saizonou

Un portfolio personnel moderne développé en HTML5 et CSS3 afin de présenter mon parcours, mes compétences, mes projets et mes services en santé publique et communication digitale.

## Aperçu

Ce portfolio met en valeur mon profil de gestionnaire de projets en santé publique et référent digital. Il a été conçu avec une interface moderne, responsive et entièrement réalisée sans framework JavaScript.

Le site comprend notamment :

- Une page d'accueil avec présentation personnelle
- Une section À propos
- Une présentation détaillée des compétences
- Les services proposés
- Les projets réalisés
- Les certificats obtenus
- Le parcours professionnel
- Un tableau récapitulatif des domaines d'intervention
- Une section contact

---

## Technologies utilisées

- CSS Grid

## Objectif du projet

Ce portfolio a été développé afin de :

- présenter mon parcours professionnel ;
- valoriser mes compétences en santé publique et communication digitale ;
- présenter mes projets personnels et professionnels ;
- faciliter la prise de contact avec les organisations, ONG et partenaires.

## Design

Le design repose sur :

- une palette de couleurs moderne ;
- une mise en page claire ;
- des cartes illustrées ;
- des effets d'ombre subtils ;
- une navigation intuitive ;
- une expérience utilisateur optimisée.
